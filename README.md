# Simple Playwright Test Agent Sandbox

## Quick Steps

1. Clone the project<br>
   `git clone https://github.com/kent-cheung-usps/playwright_test_agent.git`<br>
   
2. Chnage directory to project folder<br>
   `cd playwright_test_agent`<br>
   
3. Installs the Playwright test runner and adds it as a dev dependency<br>
   `npm install --save-dev @playwright/test`<br>
   
4. Run Sample Test<br>
   `npx playwright test`<br>
